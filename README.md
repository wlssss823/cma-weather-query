# CMA Weather Query

基于中国气象局（CMA）数据的 Python 桌面天气查询应用。

## 功能

- **实时温度查询** - 获取当前城市气温、天气状况、最高/最低温度
- **7天天气预报** - 展示未来一周的天气数据
- **24小时气温曲线** - 用折线图展示当日气温变化趋势
- **历史查询记录** - 记录并查看历史查询的城市和时间

## 技术栈

- **GUI**: tkinter
- **数据源**: 中国气象局 (weather.cma.cn)
- **网页解析**: lxml + XPath
- **数据可视化**: matplotlib
- **HTTP 请求**: requests

## 项目结构

```
.
├── weather.py          # 主程序
├── file.py             # 城市数据提取工具
├── destination.json    # 城市代码映射表
├── source.json         # 原始城市数据
├── query.json          # 查询历史记录

```

## 使用方法

```bash
# 安装依赖
pip install requests lxml matplotlib

# 运行程序
python weather.py
```

在输入框中输入城市名称（如"北京"），点击对应按钮即可查询天气。

## 运行效果

启动后显示一个简洁的查询窗口，支持四项操作：

| 按钮 | 功能 |
|------|------|
| 查询今日温度 | 显示当前气温、天气状况、最高/最低温度 |
| 查询近七日天气 | 以表格展示未来7天预报 |
| 历史查询 | 查看历史查询记录 |
| 展示24小时气温曲线 | 绘制当日气温变化折线图 |
